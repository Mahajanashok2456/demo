# demo

This project is a 3D Cinematic Travel Website built with Vite, React, TypeScript, and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd demo

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion
